## Contingency Notes

### AV Failure

| Situation | Immediate Action | Escalate to |
|-----------|-----------------|-------------|
| Projector / screen down | Switch to backup projector; AV Tech troubleshoots | AV Tech → Event Manager |
| Microphone fails | Switch to spare handheld mic immediately | AV Tech |
| Livestream drops | Notify virtual attendees via chat, attempt reconnect within 5 min | AV Tech → Event Manager |
| Full AV blackout | MC addresses room directly; break called while AV resolves | Event Manager |
| Presentation file corrupt | Use backup copy from USB; speaker uses printed notes | Speaker Liaison |

### Speaker Absent / Delayed

| Situation | Immediate Action |
|-----------|-----------------|
| Speaker running late (< 30 min) | Extend break or swap session order; MC informs audience |
| Speaker cancels same day | Move to pre-prepared panel/Q&A format; MC facilitates; use recorded content if available |
| Speaker sick on stage | Call first aid; MC takes over; break called |

### Overbooking / Venue Capacity

| Situation | Immediate Action |
|-----------|-----------------|
| More attendees than capacity | Waitlist activates; additional seating setup if possible; manage at registration desk |
| Room overfull (safety) | Redirect overflow to secondary room with livestream |
| Technical issue preventing entry (e.g. fire alarm) | Follow venue evacuation procedure; assembly point: [location] |

### Catering Issue

| Situation | Immediate Action |
|-----------|-----------------|
| Delivery late | Extend break, inform attendees; chase supplier immediately |
| Dietary requirement error | Remove item, source alternative; apologise directly to attendee |
| Insufficient quantity | Supplement with local delivery; contact Event Manager |

### Medical Emergency

1. Call emergency services: [local emergency number]
2. First aid kit location: [location in venue]
3. AED location: [location in venue]
4. Notify Event Manager immediately
5. Do not move injured person unless in immediate danger
6. Designated crew member stays with casualty until emergency services arrive
