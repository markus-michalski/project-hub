# Generic Project — Research Links

## Project Management Fundamentals

- [PMI PMBOK Guide](https://www.pmi.org/pmbok-guide-standards/foundational/pmbok) — Project management body of knowledge
- [PRINCE2 Overview](https://www.axelos.com/certifications/propath/prince2-project-management) — Structured project delivery
- [Basecamp Shape Up](https://basecamp.com/shapeup) — Lightweight product development method

## Communication & Collaboration

- [Rework (Basecamp)](https://basecamp.com/books/rework) — Lean project principles
- [Deep Work (Cal Newport)](https://www.calnewport.com/books/deep-work/) — Focus and productivity
- [Nonviolent Communication (Rosenberg)](https://www.cnvc.org/) — Effective stakeholder communication

## Decision Making

- [Amazon Working Backwards](https://www.amazon.jobs/en/principles) — PR/FAQ decision framework
- [RFC Process (IETF)](https://www.ietf.org/standards/rfcs/) — Request for Comments model for decisions
- [Architecture Decision Records (ADRs)](https://adr.github.io/) — Lightweight decision documentation
