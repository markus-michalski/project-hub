# Consulting Project

## Key Fields

- **phase**: Current engagement phase
- **go_live**: Project end / delivery date
- **budget**: Engagement value / budget

## Standard Phases

1. **Scoping** — Define scope, objectives, deliverables
2. **Analysis** — Data gathering, interviews, research
3. **Design** — Solution design, recommendations
4. **Delivery** — Presentations, workshops, handover
5. **Closed** — Engagement completed

## Standard Internal Contacts

- Lead Consultant / PM
- Subject Matter Expert(s)
- Account Manager
- Management Sponsor

## Standard External Contacts

- Client Project Manager
- Client Stakeholder(s)
- Client SME(s)
- Sub-contractor(s)

## Typical Documents

- `emails/` — Client correspondence, proposals
- `meeting-notes/` — Workshops, steering committees, interviews
- `misc/` — Deliverables, presentations, contracts

## Typical Communication Needs

- Kickoff communication to client
- Steering committee update
- Deliverable presentation follow-up
- Issue / risk escalation
- Final delivery summary
