# Event Project

## Key Fields

- **market**: Location / region
- **phase**: Current planning phase
- **go_live**: Event date
- **budget**: Event budget

## Standard Phases

1. **Planning** — Concept, venue, budget, date
2. **Preparation** — Invitations, logistics, catering, tech
3. **Promotion** — Communication, registrations
4. **Execution** — Day-of coordination
5. **Follow-up** — Thank-you notes, feedback, documentation

## Standard Internal Contacts

- Event Manager / PM
- Marketing
- Management Sponsor
- On-site Coordinator

## Standard External Contacts

- Venue
- Catering
- AV / Tech Provider
- Speaker(s)
- Agency

## Typical Documents

- `emails/` — Vendor correspondence, invitations
- `meeting-notes/` — Planning meetings, briefings
- `misc/` — Run-of-show, attendee list, budget tracker

## Typical Communication Needs

- Invitation email
- Vendor briefing
- Internal logistics update
- Speaker briefing
- Day-of coordination messages
- Follow-up / thank-you email
- Event summary / recap
