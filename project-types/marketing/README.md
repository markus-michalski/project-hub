# Marketing Project

## Key Fields

- **market**: Target market(s) / region(s)
- **phase**: Current campaign phase
- **go_live**: Campaign launch date
- **budget**: Campaign budget

## Standard Phases

1. **Briefing** — Campaign brief, objectives, target audience
2. **Concept** — Creative concept, messaging, channel strategy
3. **Production** — Content creation, asset production
4. **Review** — Internal and external review, legal/compliance sign-off
5. **Launch** — Go-live
6. **Reporting** — Performance analysis, lessons learned

## Standard Internal Contacts

- Marketing Manager / Lead
- Content Creator
- Designer
- Brand Manager
- Legal / Compliance

## Standard External Contacts

- Agency
- Media Partner
- Freelancer

## Typical Documents

- `emails/` — Agency briefs, approval requests
- `meeting-notes/` — Creative reviews, alignment sessions
- `misc/` — Briefs, assets, reports

## Typical Communication Needs

- Agency brief
- Internal stakeholder update
- Approval request
- Campaign performance summary
