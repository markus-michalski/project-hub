# Generic Project

## Key Fields

- **phase**: Current phase (free text)
- **go_live**: Target completion date
- **budget**: Budget (optional)

## Standard Contacts

- Project Lead
- Stakeholder(s)
- External Partner(s)

## Typical Documents

- `emails/` — Project correspondence
- `meeting-notes/` — Meetings and calls
- `misc/` — Any other documents

## Notes

This is a flexible catch-all type. All fields are optional.
Use more specific project types where possible for better context in summaries and communications.
